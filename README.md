# Retail Sales & Profitability Dashboard (Excel)

## Overview
This project presents an interactive Excel dashboard analyzing retail performance across sales, profit, and discount strategies. The goal was to identify inefficiencies and provide actionable insights into pricing and product performance.

## Dashboard Preview
![Dashboard](retail-dashboard.png)

## Key Features
- KPI dashboard (Sales, Profit, Margin, Orders, Discount)
- Interactive slicers (Region, Segment)
- Category and subcategory performance analysis
- Monthly sales trend visualization
- Discount vs profitability analysis

## Key Insights
- Furniture is the most profitable category overall, but performance varies significantly within subcategories.
- Tables generate consistent losses despite strong sales, indicating pricing or cost inefficiencies.
- Moderate discounting maximizes revenue without severely impacting margins.
- High discounts lead to negative profitability, suggesting diminishing returns.
- Sales show strong seasonality, with peaks in November and December.
